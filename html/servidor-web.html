Servidor Web

Realiza un servidor web en la placa microSTEAMakers que atiende a peticiones get realizadas por una pgina web a modo de panel de control. Se puede realizar en modo local (intranet) dnde las peticiones get se realizan a la ip local de la placa, o bin desde cualquier lugar (internet), redireccionando el puerto 80 del router a la ip local de la placa. En este caso las peticiones get se haran a la ip pblica del router.

Aplicacin para la placa

Atiende a diferentes peticiones get, en funcin de las mismas se mueve o emite datos de sensores. Habr que redireccionar el puerto 80 del router a la ip local de la placa que hace de servidor web.

Pgina web de control o panel (dashboard)

Se trata de una web realizada con ayuda de chat GPT a modo de panel de control del robot, se lanzarn peticiones get para moverlo y para pedirle datos de sensores para ser representados o guardados a travs de la web. Hay que modificar en la pgina web la ip en las peticiones por la ip local de la placa o bien la ip pblica del router si se ha redirigido el puerto 80 del router a la ip local de la placa.
